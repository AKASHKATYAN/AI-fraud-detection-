# AI-Based Public Fraud Detection System

This project uses machine learning to detect anomalies and potential fraud in government transactions.

## Features
- Upload CSV transaction data
- Automated feature engineering
- Isolation Forest based anomaly detection
- Interactive Streamlit dashboard

## How to Run
1. Create virtual environment
2. Install requirements
3. Run: streamlit run app.py

## Use Case
Government spending audits, procurement fraud detection, welfare leak identification.
